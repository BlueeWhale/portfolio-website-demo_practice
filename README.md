# 🌌 Space-Themed Developer Portfolio

A futuristic **3D interactive portfolio** built using React, Three.js, and GSAP.  
The portfolio is designed like a **solar system**, where each planet represents a skill.

---

## ✨ Features

- 🌍 3D Solar System UI (Three.js)
- 🚀 Animated intro screen (GSAP cinematic effect)
- 🪐 Skills as orbiting planets (React, Python, Java, SQL)
- 📸 Center profile planet with glowing ring
- ⭐ Space background with stars
- 🎯 Hover effects on skill planets
- 🖱️ Interactive 3D camera controls (OrbitControls)
- 💡 Dark futuristic UI design

---

## 🧠 Concept

- Center planet → Developer profile
- Orbiting planets → Technical skills
- Space environment → Creativity & innovation theme
- Intro animation → Cinematic ZEE5-style startup
